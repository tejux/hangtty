
=================
hangtty commands

ps      list conversations (shows 10 most recent)
ps -a   list all conversations
ps S*   list conversations starting with S 

ls      list all contacts

go <n>  open conversation (n), where (n) is index printed
        in ps output. opens a new chat session in a new tab.
        To quit session use Ctrl-C.
        At-most 6 tabs can be active at a time
        To move between sessions use Shift-Right/Shift-Left

cls     clear screen
quit    exit hangtty
exit    exit hangtty
